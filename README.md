
# 🗓️ Task Scheduler Web App

A fully functional **Task Scheduler web application** built using **HTML, CSS, and JavaScript**, designed to help users manage tasks efficiently with a clean, responsive, and modern interface.  
All tasks are stored locally using **browser localStorage**, ensuring data persistence without a backend.

##### Live hare: https://monxcode.github.io/Task-Scheduler-/

---

## 📌 Project Overview
The Task Scheduler Web App allows users to **add, edit, delete, and manage tasks** in an intuitive way.  
It is designed as a **frontend-only productivity tool**, focusing on usability, responsiveness, and smooth interaction.

---

## ✨ Core Features
- ➕ Add new tasks  
- ✏️ Edit existing tasks  
- 🗑️ Delete tasks  
- 💾 Persistent storage using `localStorage`  
- 📱 Fully responsive UI (mobile, tablet & desktop)  
- ⚡ Fast and lightweight performance  

---

## 🧠 Functional Highlights
- Tasks remain saved even after page refresh
- Real-time UI updates
- Clean task list rendering
- Simple and efficient workflow

---

## 🛠️ Technology Stack
- **HTML** – Structure & layout  
- **CSS** – Styling, responsiveness, UI polish  
- **JavaScript** – Task logic & localStorage handling  

---

## 💾 Data Management
This application uses **browser localStorage** to:
- Store tasks locally
- Retrieve tasks on page load
- Update task data dynamically
- Remove tasks permanently when deleted

No external database or backend is required.

## 🚀 How to Use
1. Open the app in your browser  
2. Add a new task using the input field  
3. Edit or delete tasks as needed  
4. Tasks are saved automatically
